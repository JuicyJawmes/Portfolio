# James Dao Portfolio

Modern static portfolio for job applications. The site is built with plain HTML, CSS, JavaScript, and local Three.js assets so it can be hosted on GitHub Pages, Netlify, Vercel, or any static file host.

## Local Preview

Open `index.html` directly, or run a small static server from this folder:

```powershell
python -m http.server 4173
```

Then visit `http://localhost:4173`.

## Deployment

Recommended options:

- Netlify: drag this folder into Netlify Drop, or connect the Git repository.
- Vercel: import the repository and deploy as a static site with no build command.
- GitHub Pages: push the repository, then enable Pages from the repository settings using the root folder.

## Files

- `index.html` - Portfolio content and structure
- `styles.css` - Responsive minimalist visual design
- `script.js` - Interactive Macintosh hub, floppy disk navigation, and contact mailto behavior
- `assets/james-dao-resume.pdf` - Resume download
- `assets/james-dao-projects.pdf` - Project archive download
